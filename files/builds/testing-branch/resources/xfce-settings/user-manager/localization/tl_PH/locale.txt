Sa lugar na ito makikita mo kung saang mga grupo kabilang ang iyong napiling user at maaari mo ring baguhin ang membership ng grupo.
Ngunit mag-ingat! Kailangan mong mapagtanto na dapat mong malaman nang eksakto kung ano ang iyong ginagawa. Ang paglalaro at pagsubok ng mga pahintulot ng user at grupo ay maaaring huminto sa iyong buong system!
Kung gusto mong gumawa o magtanggal ng user, kailangan mo lang magkaroon ng isa o mas kaunting user.
Gayunpaman, kung magbibigay ka ng higit pang mga karapatan sa isang user o nililimitahan ang mga karapatan ng isang system account o grupo, maaari mong ilagay sa panganib ang seguridad ng iyong system!
